## Overview

SIM (Security Incident Mapping) is a mini web application developed as part of the **SEN202 Software Engineering** course. The system provides a simple platform where students, staff, and landlords can report security incidents occurring on or around campus.

The aim of the project is to improve campus security by enabling quick reporting of incidents such as theft, assault, stabbing, rape, murder, and other crimes.

---

## Objectives

- Provide a simple platform for reporting security incidents.
- Encourage timely reporting of crimes.
- Improve communication between the campus community and security personnel.
- Promote a safer learning environment.

---

## Features

- Incident reporting form
- Multiple incident categories
- Location input
- Date selection
- Incident description
- Simple and responsive user interface
- Confirmation message after submission

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it with any modern web browser.
4. Fill in the form and submit an incident report.
